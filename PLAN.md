# План разработки TaskManager

## Цель проекта

Разработка консольного приложения для управления задачами (TaskManager), позволяющего пользователям эффективно организовывать и отслеживать свой список задач с возможностью сохранения данных в файл.

## Пользователи

- **Основные пользователи**: Люди, которым необходимо управлять списком задач в консольном режиме
- **Целевая аудитория**: Разработчики, студенты, пользователи Linux/Unix систем, предпочитающие консольные инструменты

## Модули приложения

1. **src/data.py** - модуль работы с хранилищем данных
   - Загрузка задач из JSON-файла (load_tasks)
   - Сохранение задач в JSON-файл (save_tasks)

2. **src/logic.py** - модуль бизнес-логики
   - Добавление задач (add_task)
   - Изменение статуса задач (change_status)
   - Редактирование задач (edit_task)
   - Удаление задач (delete_task, bulk_delete_tasks)

3. **src/ui.py** - модуль пользовательского интерфейса
   - Отображение меню (show_menu)
   - Ввод данных от пользователя (get_task_input, get_task_ids, get_new_status)
   - Вывод списка задач (display_tasks)

4. **src/main.py** - главный модуль приложения
   - Основной цикл приложения
   - Интеграция всех модулей
   - Обработка пользовательских команд

## Технологии

- **Язык программирования**: Python 3.13+
- **Формат хранения данных**: JSON
- **Стандартная библиотека**: json, typing, logging
- **Система контроля версий**: Git
- **Менеджер зависимостей**: uv (pyproject.toml)
